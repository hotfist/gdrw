# GX挑战者协会官网

欢迎来到GX挑战者协会的官方网站项目！这是一个现代化、活力充沛的社团官网，采用HTML5、CSS3和JavaScript构建。

## 📋 项目概述

**GX挑战者协会** 是一个充满热情和活力的学生社团组织，致力于为成员提供展示自我、挑战自我、相互学习的广阔舞台。

## ✨ 网站特色

### 设计风格
- **活力彩色设计**：采用红色、青绿色、黄色等活力充沛的色彩搭配
- **现代化界面**：简洁、专业的布局设计
- **响应式设计**：完美支持桌面、平板和手机设备
- **流畅动画**：增强用户体验的过渡和动画效果

### 主要功能
- 🏠 **首页** - 展示协会概况和亮点
- 📖 **关于我们** - 协会介绍、使命、发展历程和组织结构
- 👥 **成员风采** - 展示核心领导和优秀成员
- 📰 **最新动态** - 协会新闻和活动报道
- 📞 **联系方式** - 在线表单、社交媒体和FAQ

## 🚀 快速开始

### 1. 打开网站
直接在浏览器中打开 `index.html` 文件即可查看网站。

### 2. 本地开发
如果需要本地开发，建议使用以下方法之一：

**方法一：VS Code Live Server**
```bash
1. 在VS Code中安装 Live Server 扩展
2. 右键点击 index.html
3. 选择 "Open with Live Server"
```

**方法二：Python HTTP服务器**
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000

# 然后访问 http://localhost:8000
```

**方法三：Node.js http-server**
```bash
npm install -g http-server
http-server
```

## 🔧 主要功能实现

### 1. 响应式导航菜单
- 桌面端：横向菜单
- 移动端：汉堡菜单
- 点击链接自动关闭菜单

### 2. 表单验证
- 必填项检查
- 邮箱格式验证
- 提交后显示成功提示

### 3. FAQ手风琴
- 点击问题展开/收起答案
- 同时只展开一个问题
- 平滑的开关动画

### 4. 回到顶部按钮
- 滚动超过300px时出现
- 点击平滑滚回顶部
- 鼠标悬停时有动画效果
